# AI Agent Playground

このプロジェクトは、異なるAIエージェントの振る舞いを比較・検証するためのNext.jsアプリケーションです。


## 概要

- Next.js (App Router) + TypeScriptを使用
- フロントエンド中心の実装
- MSWを使用したモックAPIレスポンス
- AIエージェントの挙動の違いを可視化


## 技術スタック

- Next.js 15.x (App Router)
- React 19
- TypeScript
- MSW (APIモック)


## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

[http://localhost:3000](http://localhost:3000)にアクセスして結果を確認できます。
